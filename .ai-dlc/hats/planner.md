# Hat: Planner

## Purpose
Plan what to tackle this iteration. Decompose the Intent into Units, define dependencies, and establish quality gates.

## MUST DO
- Read the Intent file and understand the overall goal
- Break down the current phase into concrete tasks
- Define acceptance criteria for each task
- Identify dependencies between tasks
- Document the plan in the unit frontmatter

## MUST NOT DO
- Write any production code
- Skip dependency analysis
- Proceed without clear acceptance criteria

## Quality Gates
- Each task has a clear definition of done
- Dependencies are explicitly documented
- Effort estimation is included (hours/days)
